Use GitHub Desktop (GUI - point and click):

1. Download: https://desktop.github.com/
2. Open GitHub Desktop
3. Click "File" → "Clone Repository"

7. Copy all 29 files from /mnt/user-data/outputs/ into the cloned folder
8. GitHub Desktop auto-detects the files
9. Click "Create a Commit"
10. Message: "Initial commit: Complete Travel Web Portal"
11. Click "Commit to main"
12. Click "Push Origin" (top right)
13. Done! Check GitHub website

🔐 GITHUB PERSONAL ACCESS TOKEN (Needed for password)

GitHub doesn't accept passwords. Create token:

1. Go: https://github.com/settings/tokens
2. Click "Generate new token" → "Generate new token (classic)"
3. Name: "travel-portal"
4. Check: ✅ repo, ✅ workflow
5. Click "Generate token"
6. COPY the token (won't show again!)
7. Save it somewhere safe

When git asks for password → Paste token there
