⚡ 5-STEP DEPLOYMENT:

STEP 1: Clone to Your Computer
   Download all files from /mnt/user-data/outputs/
   Or initialize a fresh git repo and add the files

STEP 2: Configure APIs
   Get Claude API key from console.anthropic.com
   Add it in Settings tab > Claude API Configuration

STEP 3: Push to GitHub
   git init
   git add .
   git commit -m "Initial commit: Travel web portal"
   git branch -M main
   git remote add origin https://github.com/mosscow/travel-web-portal.git
   git push -u origin main

STEP 4: Enable GitHub Pages (FREE HOSTING)
   Go to GitHub repo Settings > Pages
   Select "main" branch as source
   Wait 2-3 minutes
   Your site is live at: https://mosscow.github.io/travel-web-portal/

STEP 5: Share & Enjoy!
   Send the link to friends
   Let them plan their Italy trip!

═══════════════════════════════════════════════════════════════════════════════

🔑 GETTING CLAUDE API KEY (5 minutes):

1. Visit https://console.anthropic.com
2. Sign in with GitHub or email
3. Click "API Keys" in left sidebar
4. Click "Create new API key"
5. Copy the key (starts with "sk-ant-")
6. Paste in app Settings tab
7. Click "Save API Key"
8. Click "Test Connection"
9. Done! You're ready to chat with Claude
